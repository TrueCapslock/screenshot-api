Screenshot API

Capture screenshots of any web page or render HTML to an image. Compare against baselines, detect visual diffs, and describe changes with AI — all through a simple REST API.

Ta skjermbilder av alle nettsider eller render HTML til bilde. Sammenlign med referansebilder, oppdag visuelle forskjeller og beskriv endringer med AI — alt via et enkelt REST API.

Features

Funksjoner

📷

URL Screenshots

URL-skjermbilder

Capture any publicly accessible URL. Supports custom viewport, mobile emulation, full-page capture, and dark mode.

Fang alle offentlig tilgjengelige URL-er. Støtter egendefinert visningsport, mobil-emulering, fullsidefangst og mørk modus.

📝

HTML Rendering

HTML-rendring

Render raw HTML markup to an image — no web server needed. Perfect for generating share cards, emails, or previews.

Render rå HTML-kode til bilde — ingen webserver nødvendig. Perfekt for delingskort, e-poster eller forhåndsvisninger.

🔄

Visual Comparison

Visuell sammenligning

Compare screenshots against baselines with pixel-level diff detection. Get diff percentage, pixel count, and visual overlay.

Sammenlign skjermbilder mot referansebilder med pikselnøyaktig diff-deteksjon. Får diff-prosent, pikselantall og visuelt overlay.

✨

AI Describe

AI-beskrivelse

AI-powered description of visual differences between two screenshots. Pro+ tier.

AI-drevet beskrivelse av visuelle forskjeller mellom to skjermbilder. Pro+ nivå.

⚡

Async Capture

Asynkron fangst

Queue screenshots via BullMQ for non-blocking capture. Poll for results or receive a webhook when complete.

Legg skjermbilder i kø via BullMQ for ikke-blokkerende fangst. Hent resultater eller motta webhook når ferdig.

🔒

API Key Auth

API-nøkkel-autentisering

Secure bcrypt-hashed API keys with tier-based rate limiting and monthly usage quotas. Stripe subscription billing.

Sikre bcrypt-hashede API-nøkler med nivåbasert rate-begrensning og månedlige kvoter. Stripe-abonnementsfakturering.

Quick Start

Hurtigstart

# Capture a screenshot # Ta et skjermbilde
curl -X POST https://screenshots.hartvig.info/v1/screenshot \
  -H "x-api-key: sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com","format":"png","fullPage":true}' \
  -o screenshot.png

# Compare against a baseline # Sammenlign med et referansebilde
curl -X POST https://screenshots.hartvig.info/v1/compare \
  -H "x-api-key: sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com","baseline_id":"your-baseline-uuid"}'
